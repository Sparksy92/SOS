# Reference Project Comparison Matrix

## WROLPi Comparison
- **Usable Concepts:** Raspberry Pi targets, one-time pad encrypter/decrypter, universal search.
- **Do Not Use:** GPLv3 source code merges, auto-starting unvetted wireless hotspots.

## Project N.O.M.A.D. Comparison
- **Usable Concepts:** Kiwix/ZIM content manager architectures, local vector semantic matching.
- **Do Not Use:** Automatic updates, internet checks, or unauthenticated LAN access configurations.

## Internet-in-a-Box Comparison
- **Usable Concepts:** Wikipedia Medical Encyclopedia content packs, Khan Academy Lite integrations.
- **Do Not Use:** Monolithic SD-card images or auto-exposure of local server files.
