Core Operations Modules

🗺️

Tactical Command & Operations

Launch structured missions from offline templates. Track patrollers, coordinates, and checklists. Log events on an active timeline and export reports in PDF and Markdown formats.

🧠

Local AI Assistant & RAG

Extract manual instructions and guidelines locally using the R.A.N.G.E.R. conversational agent. Powered by an in-process vector store (HNSWLib) and Ollama. Runs fully offline.

🥫

Pantry & Water Calculators

Log inventory levels, audit container capacities, track daily caloric consumption parameters, and calculate storage runtime survival metrics through specialized estimators.

📚

Library Vetting Ledger

Scan local staging folders, approval histories, and licenses. Reconcile missing topic guides using the Content Gap Analyzer before indexation.

📡

Meshtastic Integration

Connect Heltec/T-Beam GMRS LoRa nodes over off-grid LAN. Monitor signal packet pathways, check inpatrol logs, and sync waypoints to your Leaflet topographical map.

⚡

Off-Grid Diagnostics

Calculate usable solar panel capacity outputs, LiFePO4 battery deep discharge protections, and appliance load estimates inside the app.
